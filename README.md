# Snippets - 文本片段管理工具

## 技术栈

- 编程语言：AutoHotkey v2
- 数据存储：JSON 文件（snippets.json）
- 运行环境：Windows

## 安装使用

### 方式一：直接运行脚本
1. 安装 [AutoHotkey v2](https://www.autohotkey.com/)
2. 双击运行 `snippets.ahk`

### 方式二：运行编译版本
直接双击 `snippets.exe`，无需安装 AutoHotkey

## 功能

- **快捷键呼出**：默认 `Ctrl+`` ` 呼出/隐藏窗口
- **搜索**：实时搜索标题和内容
- **复制**：双击条目自动复制内容到剪贴板
- **新增**：点击 `+` 按钮或列表末尾空白行展开编辑面板
- **编辑**：单击条目展开底部编辑面板修改内容
- **删除**：编辑面板中点击删除按钮
- **撤销**：`Ctrl+Z` 撤销删除操作，最多20步
- **保存快捷键**：编辑时按 `Enter` 保存，`Shift+Enter` 换行
- **开机自启**：右键托盘图标 → 开机自启 切换开启/关闭
- **窗口拖动**：拖动顶部标题栏移动窗口
- **自适应宽度**：拖动窗口边缘横向拉伸，内容自适应

## 数据文件

数据保存在 `snippets.json`，与程序同目录，可手动备份。
